# Inspect Selected Widget

シミュレーターで選択したウィジェットを調査・修正するワークフロー。

## 手順

1. **ウィジェット選択モードを有効化**
   - `set_widget_selection_mode(enabled: true)` を実行
   - ユーザーに「シミュレーターでウィジェットをタップして選択してください」と伝える

2. **ユーザーの選択を待機**
   - ユーザーが「選択した」「ok」「y」などと報告するまで待つ

3. **選択されたウィジェットを取得**
   - `get_selected_widget()` で選択情報を取得
   - ウィジェットの種類、プロパティ、ソースコード位置を確認

4. **関連コードを調査**
   - ソースコード位置からファイルを読み込み
   - ウィジェットの実装を確認
   - 親ウィジェットや関連するProviderも必要に応じて調査

5. **修正提案または実行**
   - ユーザーの指示に基づいて修正を提案/実行
   - 修正後は `hot_reload()` で即座に反映

6. **選択モードを無効化**
   - 作業完了後 `set_widget_selection_mode(enabled: false)` を実行

## 使用するDart MCPツール

- `set_widget_selection_mode` - 選択モードのON/OFF
- `get_selected_widget` - 選択されたウィジェット情報取得
- `hot_reload` - 変更の即時反映

## 注意事項

- DTD接続済みの状態で実行すること
- 選択モードはアプリのUI操作を妨げるため、調査完了後は必ずOFFにする
